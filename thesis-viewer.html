About This Model

This is an interactive 3D reconstruction of a an Ifugao bale house scale model created using photogrammetry. You can explore the model freely by rotating, zooming, and panning around it.

How to Navigate
Rotate Left-click and drag anywhere on the model
Zoom In / Out Scroll the mouse wheel up or down
Pan / Move View Right-click and drag to shift the view
Reset View Click the Reset button to go back to the default camera position
Toolbar Buttons
WF
Wireframe Shows the mesh structure of the model. Tap again to turn off.
⌂
Reset Camera Returns to the default view if you get lost navigating.
CLR
Color Toggle Switch between the original texture colors and a plain grey view.
☀
Brightness Adjust how bright or dark the model appears using the sliders.
+
Load File Load a different 3D file. Supports .glb, .gltf, and .ply formats.
