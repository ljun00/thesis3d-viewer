
# thesis3d-viewer

A web-based 3D model viewer built with HTML, CSS, and JavaScript. AI assisted coding

## What is this?

This project is a 3D viewer that allows you to view and interact with 3D models directly in your web browser. No installation or special software needed!

## How to use it

Click the link below to open the 3D viewer:

**[3D Viewer](https://ljun00.github.io/thesis3d-viewer/thesis3d-viewer.html)**

### What happens when you click the link?

1. Your browser will load the 3D viewer page
2. The 3D model will automatically download from this repository
3. You can interact with the model:
   - Rotate the model by dragging your mouse
   - Zoom in and out using your mouse scroll
   - The viewer will render the model in 3D

## Project Structure

- `thesis3d-viewer.html` - The main HTML file containing the viewer interface, styling, and JavaScript code
- `models/` - Folder containing the 3D model files
